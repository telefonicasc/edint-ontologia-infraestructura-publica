Mappings RML para transformar datos de infraestructura municipal a RDF (EDINT Infraestructura)

Release: 11-03-2026

Version:
1.0.0
Authors:
Jhon Toledo
Description:
Este conjunto de mappings RML transforma los datos de infraestructuras públicas de un municipio a RDF, utilizando la ontología EDINT Infraestructura. El objetivo es generar un grafo RDF interoperable para representar las entidades principales de la infraestructura pública municipal, como equipamientos educativos, sociales, culturales, deportivos, aparcamientos y estaciones de bicicletas compartidas, orientado a los usos habituales de las entidades locales.
